# 🐾 반려동물 통합관리 앱

> 강아지·고양이 보호자를 위한 올인원 반려동물 관리 웹 서비스

<br>

## 📌 프로젝트 소개

반려동물 초보 보호자부터 숙련자까지, 급식 관리·건강 체크·동물병원 찾기·AI 병명 예측까지 하나의 앱에서 해결할 수 있는 통합관리 서비스입니다.

- **활동기간**: 2026.03 ~ 2026.06
- **소속**: 연성대학교 컴퓨터소프트웨어과 클라우드 융합컴퓨팅 / 캡스톤 디자인 1

## ✅ 기능 목록

- [ ] ① 초보 보호자 가이드 — 종류별 사육 방법, 준비물, 주의사항 단계별 안내
- [ ] ② 급식 알림·급여량 계산기 — 푸시 알림, 나이·체중 기반 급여량 자동 계산
- [ ] ③ 증상 기반 건강 체크 — 증상 입력 시 의심 질환 및 병원 방문 권고
- [ ] ④ 동물병원·보호소 찾기 — 현재 위치 기반 지도 표시 및 상세 정보
- [ ] ⑤ 위험 음식 검색 — 유해 식품 검색, 위험도·증상 정보 제공
- [ ] ⑥ 산책 가능 여부 판단 — 체중 + 실시간 날씨 데이터 연동
- [ ] ⑦ 훈련 가이드 — 기초 훈련부터 행동 교정까지 단계별 방법
- [ ] ⑧ AI 병명 예측·병원 추천 — Gemini API 기반 증상→병명 예측
